# Vue-vant
使用Vue和vant搭建一个APP

1. 基于Vue框架，组件`vant`和库MUI的配合使用搭建一个简易的APP
    - 需要安装最新版`webpack4.X`版本
    - 需要自行下载`m-ui`第三方框架
    - 其余配置 参照 `webpack.config.js`
2. vue请求数据为`vue-resource` 
3. clone 到本地 请先使用你` npm install `,安装对应的包，启动脚本使用`webpack-dev-server`
